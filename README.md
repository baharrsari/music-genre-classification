# Music Genre Classification

Bu proje, GTZAN veri seti kullanılarak müzik türü sınıflandırması yapmak için hazırlanmıştır.

## Proje hedefleri
- Birden fazla machine learning algoritmasını karşılaştırmak
- Aynı modelin farklı hiperparametreli sürümlerini karşılaştırmak
- Çok sayıda performans metriği kullanmak
- Sonuçları görselleştirmek ve yorumlamak

## Klasör yapısı
- `notebooks/` : Ana notebook
- `src/` : Tüm Python modülleri
- `data/` : Ham, ara ve işlenmiş veriler
- `models/` : Eğitilmiş modeller ve sonuçlar
- `reports/` : Grafikler ve tablolar
